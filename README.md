# mermaid-converter

## 開発方法
- clone
    ```sh
    $ git clone git@github.com:SOTAkkkk/mermaid-converter.git
    ```
    ```sh
    $ cd mermaid-converter
    ```
- npmインストール
    ```sh
    $ npm install
    ```
- 起動
    ```sh
    $ npm run start
    ```
- パッケージ化
    ```sh
    $ npm run make
    ```
